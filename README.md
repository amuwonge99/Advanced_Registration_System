Go
==

This repo is to store a set of labs to help get started with Go. The offical guide to Go is also very useful for learning the language

That is available [here](https://tour.Go.org/welcome/1)

GETTING STARTED
===============

Go should be installed using the official guidance available from [here](https://go.dev/doc/install)

To check that it has been installed correctly run the following command

```sh
$ go version
go version go1.24.3 darwin/amd64
```

(Though, your version may vary slightly)
